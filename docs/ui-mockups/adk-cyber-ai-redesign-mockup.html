What this mock changes

  • Brain bar — Provider + Model + quota always above the input
  • Settings drawer — Account, Local LLM, Firewall (not a permanent rail)
  • Collapsible left nav — Recent focus + collapsed product groups
  • Single New chat — left rail only
  • Explicit Focus — product selection explained in the bar
  • Thinking chip + copy/regenerate on assistant messages

How to use this file

  • Open in Edge/Chrome (double-click or drag into browser)
  • Hover dashed outlines for change callouts
  • Change Provider → Model list + privacy line update
  • Click Settings or avatar for drawer
  • Click Send to show a sample thread
  • Toggle Current pain map for before/after density

Out of scope in this mock

  • Real bridge / streaming / auth
  • Full light-theme pass
  • iOS compact layout (same brain bar pattern applies)
  • Pixel-perfect parity with production CSS

Suggested build order

  • P0 Brain bar + wire to existing /api/settings
  • P0 Settings drawer (move provider + local fields)
  • P1 Collapsible left nav + recents
  • P1 Stop / copy / regenerate / thinking UI
