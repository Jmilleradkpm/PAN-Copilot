Welcome to ADK Cyber AI

    
A few things worth knowing before you paste your first config:

    
    
      
  • Runs locally: the app lives on your Windows PC; chats are stored at ~/.pan_copilot/conversations_v3/
    • 
      
  • Redacted on device: passwords, PSKs, API keys, and private keys are stripped before cloud inference
    •

    • 
      
  • Local plan: switch to your own LLM in Settings — nothing leaves your machine

🔒 What gets sent when you submit a config

    
Before you send your first config or screenshot, here's exactly what happens on cloud plans:

    
    
      
  • Auto-redacted before sending: passwords, shared secrets, pre-shared keys, API keys, private keys
    • 
      
  • Still sent (so the AI can reason): network topology, IP addresses, zone architecture, rule structure
    • 
      
  • Screenshots: not redacted — crop or blur secrets before attaching
    • 
      
  • ADK Cyber gateway: redacted requests pass through in transit only (not stored); your account server sees email, session, and usage only
    • 
    

    
Recommendation: paste only the section relevant to your question — a specific policy rule, BGP neighbor config, or IKE profile — rather than a full show config running export.
